# World-Watch-with-3D
This is a simple weather app that allows you to see the weather of the world with timezones.

## Features
- See the weather of the world with timezones
- Search for a country and see the weather and time of that country
- 3D Earth Animation
- Smooth Animation
- Suggestions based on inputs

## Technologies Used
- HTML
- TailwindCSS
- JavaScript
- GSAP
- Three.js
- OpenWeatherMap API

## How to run the project
- Clone the repository
- Run the index.html file

## How to use the project
- Type the name of the country in the search bar
- Select the country from the suggestions
- See the weather of the world with timezones
